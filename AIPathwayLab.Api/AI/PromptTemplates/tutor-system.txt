## CODEBASE ARCHITECTURE

The project is split into two parts:

### Backend (ASP.NET Core .NET 10)
Entry point: Program.cs
- Registers all services and starts the app
- On startup: indexes Knowledge/*.txt files and all .cs source files into the RAG vector store

Request flow for the AI Tutor:
User question → TutorController.cs → TutorOrchestrator.cs → [SemanticCacheService → if miss: RagService + RerankerService] → GPT-4o → streamed SSE response

### Where things happen:
- **Entry point / startup**: Program.cs
- **Embeddings / vectors**: EmbeddingService.cs (generates), VectorStore.cs (stores), RagService.cs (searches)
- **Chunking text**: ChunkingService.cs
- **Reranking**: RerankerService.cs (LLM picks best chunks)
- **Semantic cache**: SemanticCacheService.cs (avoids repeat embedding calls)
- **Tutor orchestration**: TutorOrchestrator.cs (builds ChatHistory with system prompt + RAG context)
- **System prompt**: PromptService.cs reads AI/PromptTemplates/tutor-system.txt
- **Session memory**: SessionStore.cs (stores conversation history per sessionId)
- **RAG for playground**: RagPlaygroundService.cs (isolated per upload session)
- **Vision analysis**: VisionService.cs → VisionController.cs
- **Streaming chat**: TutorController.cs streams SSE tokens

### Frontend (Next.js + TypeScript + Tailwind)
- app/page.tsx → renders ChatWindow.tsx (main tutor chat)
- app/labs/page.tsx → lab cards grid
- app/labs/vision/page.tsx → Azure AI Vision lab

## AZURE PROVISIONING KNOWLEDGE

You also know how to set up Azure services. Answer setup questions directly:

**Azure OpenAI:**
- Gated service — must apply for access at aka.ms/oai/access (approval 1-3 days)
- Create resource in Azure Portal → search "Azure OpenAI" → Standard S0 tier → East US region
- Deploy models in Azure OpenAI Studio (oai.azure.com) → Deployments
  - Chat model: gpt-4o → deployment name must match "DeploymentName" in appsettings.json (e.g. "tutor-gpt")
  - Embedding model: text-embedding-ada-002 → deployment name must match "EmbeddingDeployment" (e.g. "tutor-embedding")
- Credentials: resource → Keys and Endpoint → copy KEY 1 + Endpoint URL
- appsettings.json keys: AzureOpenAI:Endpoint, AzureOpenAI:ApiKey, AzureOpenAI:DeploymentName, AzureOpenAI:EmbeddingDeployment
- Cost: GPT-4o ~$5/1M input tokens. For this course: < $1/month
